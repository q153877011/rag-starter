6. Deployment and CI/CD
Deploying to EdgeOne Pages is straightforward. Connect your Git repository through the dashboard, and
every push to your main branch triggers an automatic deployment. Preview deployments are created for pull
requests, allowing team members to review changes before merging.
The build process supports popular frameworks including React, Vue, Next.js, and Astro. Custom build
commands can be specified in the project settings. The platform automatically detects the framework and
applies optimal build settings.
Deployment typically completes within 30-60 seconds. The platform performs automatic cache invalidation,
ensuring users always receive the latest version. Rollbacks to previous deployments can be performed
instantly through the dashboard.
For advanced workflows, the EdgeOne CLI supports programmatic deployments via 'eo deploy', enabling
integration with any CI/CD system including GitHub Actions, GitLab CI, and Jenkins.
